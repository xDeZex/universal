# A Locked Workout can be corrected but not extended

A Workout becomes Locked when finished: its Sets and Exercise Entries can still be edited or deleted, but no new Set or Exercise Entry can be added to it (see #123). The alternative would be to either forbid all editing once finished (matching how it renders today) or leave it fully open to new logging — but forbidding all editing would make ordinary mistakes (a wrong weight, an extra Set) permanently unfixable once Finish is pressed, while allowing new entries would blur Finish's meaning as "this Workout is done" into something reversible. Drawing the line at correct-not-extend keeps Finish meaningful while still letting the user fix what's already there. This also means Locked is not a single on/off read-only mode: an in-progress Workout allows both adding and correcting, a Locked one allows only correcting (see #135, #136).
